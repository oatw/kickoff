简介

Kickoff是以数据驱动思想设计并实现的任务管理组件。包涵了计划排布所需的通用功能特性， 且提供了丰富的定制化接口，以支持开发人员在复杂的业务场景下深度定制。

目前，第一个测试版0.2.0已正式发布，可通过 首页Demo 及 在线示例 试用了解核心特性及定制化接口。 在正式版发布之前，内部实现和公共API（尤其是源码中暴露的但未公开在文档中的API）可能发生较大变化。 因此，暂不推荐在生产环境中使用。

Kickoff的源码使用 Coffeescript 和 Sass 编写， 工程化脚本基于 Ruby Gem Thor 构建，UI组件基于Luda / Kernel Engine 构建。欢迎大家 参与贡献。
