配置管理

在使用Kickoff过程中，所有的配置都应集中通过kickoff构造函数，或其实例方法configure管理。 在Kickoff实例化完成以后，可以通过实例的config属性访问到所有的配置项， 这些配置项会按需同步到了不同的模型中。
